Scientific scope

A static homepage that still feels like a real conference site.

The public IAUS406 site centers on a symposium about the future landscape of astrophysical transients in Turku, Finland, with linked sections for programme, speakers, venue, deadlines, grants, and registration.

This static version preserves that structure while removing all WordPress and PHP dependencies so it can run on GitHub Pages.
